# lightsail

- AWSの初心者向け的な
  - コンソールの日本語対応が完璧でびっくりした
  - コンソールはAWSコンソールのデザインとは全然別
  - lightsail でインスタンスを立てても EC2 コンソールでは見えないっぽい
  - レンタルサーバーの利用者向けに、AWSでも同じような操作を実現できるよ、って謳っているものだと思っている
- インスタンスの起動だけすることもできるが、そのあとWordpressやGhostを立てることもできる
  - linodeみたいに。
  - bitnami がベースらしい
  - bitnami は有名な docker イメージ作っているところって認識だったが、そもそもこういうふうにパッケージングして配布/起動できるようにするのが目的らしい
- ミニマムで5ドル
  - 今や Public IP は有料であり EC2 ではその分のお金もかかるので、それを考えるとちょっと安いかも。
  - 一方で拡張性がないのでプロダクションは難しいと思う
