# arktype & arkregex

### arktype
- arktype 自体は、zod に似たライブラリ
- zod との違いは書き心地だと感じた。
  - https://medium.com/@ruverd/why-use-arktype-instead-of-zod-08c401fd4f6f
  - z.xx() てネストせずに、トップレベルに type() をおくだけでいける
- 直感的
- だが全部のユースケースをカバーできるかは不明。ちょっといびつに感じるかも

Links
- https://arktype.io
- https://medium.com/@ruverd/why-use-arktype-instead-of-zod-08c401fd4f6f
- https://github.com/arktypeio/arktype

### arkregex
- よくできている。
- JS で正規表現ってちょっと面倒な印象。クラスだし。
- タイプにするという意味合いで arkregex を使ってもいいかなと思った。
