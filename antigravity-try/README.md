# antigravity

- Google の AI IDE
- Google アカウントでのログインが必要
- VS Code の fork
- Calude Code をエディタで実行している感じ。GitHub Copilot と変わらない。そんなに飛躍はしていない。
- なんか実装計画を立てるステップがありそう。だからむやみにファイルを変更しているわけではない
  - が出力物について GitHub Copilot との違いはあんま感じない
- 今のところ無料らしい
  - https://antigravity.google/pricing
- agent manager というのは github copilot のセッションの画面に似ている。
  - issue をアサインして pr を作ってもらっている時に、内部でなんかやっているのを見れる場所
- AIがブラウザも開けるようになるらしい
- 全体的に新しい概念はない
  - claude や github copilot がいろいろ導入した機能を Google なりに整理
  - エディタという形式にまとめた感じに見える
- 情報の扱いが不明なので実務で使えるかは微妙

## Install
```bash
brew install --cask antigravity
```
